Commands:
help - Show all available commands
close - Close the program
logout - Log out of the program
fullscreen - Toggles fullscreen (F11 works as well)
clear - Clear the screen
username view|change - View or change your username
password view|change - View or change your password
notes - Write and read your own notes

System Controls:
-back - Exit to main menu

Tip: You only have to write the first two letters of the commands
